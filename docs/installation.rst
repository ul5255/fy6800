============
Installation
============


Stable release
--------------

To install fy6800, run this command in your terminal:

.. code-block:: console

    $ pip install fy6800

This is the preferred method to install fy6800, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for fy6800 can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/ul5255/fy6800

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/ul5255/fy6800/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/ul5255/fy6800
.. _tarball: https://github.com/ul5255/fy6800/tarball/master
